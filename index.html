HyperCore Privacy Policy

Effective Date: July 14, 2026

Information We Collect

HyperCore may process Discord User IDs, Guild IDs, Channel IDs, Role IDs, Message IDs, server configuration data, backup data, and command interactions only when required to provide its services.

Message Content

HyperCore only accesses message content when required to execute commands, moderation features, or server management functions requested by authorized administrators.

HyperCore does not use message content to train AI or machine learning models.

Backups

HyperCore allows authorized server owners to create and restore backups of channels, roles, permissions, categories, emojis, and server settings.

Data Storage

Only the minimum amount of information necessary to provide backup and management features is stored securely.

Data Sharing

HyperCore never sells or shares user information with third parties unless required by law.
